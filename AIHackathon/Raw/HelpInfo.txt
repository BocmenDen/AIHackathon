Привет на хакатоне! Этот бот оценивает модели ИИ. Ваша задача: предсказать значения столбца Win, и стать первыми в рейтинге. Общий рейтинг будет строиться на основании метрики ROC AUC вашей модели. Будьте готовы что данная метрика вычисленная у вас локально скорее всего будет отличаться с тем что покажет вам бот это связанно с тем что у нас заготовлены другие данные для тестирования моделей.

Общие требования к нормализации данных:

• Категориальные:
  • True/False в 1/0
  • teamId: 100 в 0, 200 в 1
  • win: Win в 1, Fail в 0
• Удаление: Удалите столбец dominionVictoryScore.
• Числовые: Примените Min-Max Scaling.

Важно: Сохраняйте исходную последовательность столбцов!

Поддерживаемые библиотеки: В данный момент бот поддерживает модели Keras  и XGBoost. Для других библиотек, пожалуйста, свяжитесь с @bocmenden и предоставьте скрипт загрузки модели и выполнения прогнозирования. Не забывайте что некоторые библиотеки к примеру Keras требовательны к расширению файлов, убедитесь, что ваша модель сохранена с нужным расширением (для Keras к примеру чаще всего это .h5).

Чтобы выгрузить свою обученную модель в файл для XBoost вы можете использовать следующий шаблон:
```
import xgboost as xgb

# ... ваш код обучения модели ...

xgb.Booster.dump_model(model, 'FILE_NAME.json')
```
Чтобы выгрузить свою обученную модель в файл для Keras вы можете использовать следующий шаблон:
```
from tensorflow import keras

# ... ваш код обучения модели ...

model.save("my_keras_model.h5")
```
